# nextjs-k8s-strategy

## setup && development

```
$ yarn install
$ yarn dev
```

## docker build

```
$ docker build -t hiro08/nextjs-k8s-strategy .
```

## docker run 

```
$ docker run -d -p 3000:3000 hiro08/nextjs-k8s-strategy:latest
```

## docker registory

```
$ docker pull hiro08/nextjs-k8s-strategy:latest
```

Open localhost:3000
